GlucoVigil Health Analytics - Use Case Diagram

Use Case Description

The diagram shows three main actors:

  • End User: Primary system user who manages their health data
  • Healthcare Provider: Medical professionals who access patient data
  • System Administrator: Technical personnel managing the system

Key use cases include:

  • User Registration and Authentication
  • Health Data Management
  • Risk Assessment
  • Progress Tracking
  • Report Generation
  • System Maintenance
