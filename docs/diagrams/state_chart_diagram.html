GlucoSmart Health Analytics - State Chart Diagram

This state chart diagram illustrates the different states a user account can be in within the GlucoSmart system, and the transitions between these states based on events and conditions.

User Account Lifecycle States

The state chart shows the progression of a user account through various states:

  • Unregistered: Initial state before user creates an account
  • Registered (Unverified): Account created but email not verified
  • Verified: Email verification completed
  • Profile Incomplete: Basic profile information pending
  • Profile Complete: User has provided necessary profile information
  • Health Data Incomplete: Waiting for health metrics input
  • Health Data Complete: Required health data provided
  • Risk Assessed: Risk calculation completed
  • Inactive: User hasn't logged in for extended period
  • Suspended: Account temporarily disabled
  • Terminated: Account permanently closed

Key Transitions

The diagram illustrates important transitions between states:

  • Registration: Unregistered → Registered (Unverified)
  • Email Verification: Registered (Unverified) → Verified
  • Profile Completion: Profile Incomplete → Profile Complete
  • Health Data Entry: Health Data Incomplete → Health Data Complete
  • Risk Assessment: Health Data Complete → Risk Assessed
  • Inactivity Timeout: Any Active State → Inactive
  • Reactivation: Inactive → Last Active State
  • Account Closure: Any State → Terminated

Guards and Conditions

The state transitions are controlled by guards and conditions:

  • Email Verification Required: Controls transition to Verified state
  • Minimum Profile Info Required: Controls transition to Profile Complete
  • Sufficient Health Data Required: Controls transition to Health Data Complete
  • Inactivity Timeout (90 days): Triggers transition to Inactive
  • Policy Violation: Triggers transition to Suspended

Composite States

The diagram shows nested states within larger state groups:

  • Active Account: Contains multiple sub-states (Verified, Profile stages, etc.)
  • Onboarding Process: Groups the progressive states of initial account setup
